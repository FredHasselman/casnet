# Check a Recurrence Matrix

Check a Recurrence Matrix

## Usage

``` r
rp_checkfix(
  RM,
  checkS4 = TRUE,
  checkAUTO = TRUE,
  checkSPARSE = FALSE,
  checkTSPARSE = FALSE,
  fixS4 = FALSE,
  fixAUTO = TRUE,
  fixSPARSE = FALSE,
  fixTSPARSE = FALSE
)
```

## Arguments

- RM:

  RM

- checkS4:

  checkS4

- checkAUTO:

  checkAUTO

- checkSPARSE:

  checkSPARSE

- fixS4:

  fixS4

- fixAUTO:

  fixAUTO

- fixSPARSE:

  fixSPARSE

## Value

A checked and/or fixed matrix
