# Get indices of matrix diagonals, rows, or columns

Get indices of matrix diagonals, rows, or columns

## Usage

``` r
mat_ind(Xlength, Ylength, index, diagonal = FALSE)
```

## Arguments

- Xlength:

  X dim

- Ylength:

  Y dim

- index:

  index of diagonal, row or column

- diagonal:

  diagonal

- horizontal:

  horizontal

## Value

list
