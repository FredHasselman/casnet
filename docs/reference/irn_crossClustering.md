# Cross CLustering Coefficient

Cross CLustering Coefficient

## Usage

``` r
irn_crossClustering(Ax, Ay, Axy, local = FALSE, global = TRUE)
```

## Arguments

- Ax:

  Adjacency matrix x

- Ay:

  Adjacency matrix y

- Axy:

  Adjacency matrix xy

- local:

  local CC

- global:

  global CC

## Value

list object with local and global CC
