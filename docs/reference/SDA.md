# Standardised Dispersion Analysis

Adaptation from discontinued package `fractal` by William Constantine
and Donald Percival.

## Usage

``` r
SDA(x, front = FALSE, scaleS)
```

## Arguments

- x:

  x

- front:

  front

- scaleS:

  If not `NA`, it should be a numeric vector listing the scales on which
  to evaluate the detrended fluctuations. Arguments
  `scaleMax, scaleMin, scaleResolution` will be ignored (default = `NA`)

## Value

output
