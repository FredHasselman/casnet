# Get embedded series for rp() and rqa()

Get embedded series for rp() and rqa()

## Usage

``` r
rqa_getSeries(
  y1,
  y2 = NULL,
  emDim = 1,
  emLag = 1,
  chromatic = FALSE,
  doEmbed = TRUE
)
```

## Arguments

- y1:

  y1

- y2:

  y2

- emDim:

  emDim

- emLag:

  emLag

- chromatic:

  chromatic

- doEmbed:

  doEmbed

## Value

embedded series
