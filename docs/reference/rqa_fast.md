# Fast rqa

Fast rqa

## Usage

``` r
rqa_fast(
  y1,
  y2 = NA,
  index,
  theiler,
  emRad,
  symmetrical = TRUE,
  diagonal = FALSE,
  method = "Euclidean",
  withSplits = FALSE,
  splitSize = 2048
)
```

## Arguments

- y1:

  y1

- y2:

  y2

- index:

  row

- theiler:

  theiler

- emRad:

  threshold

- symmetrical:

  symmetrical

- diagonal:

  diagonals

- method:

  distance method

- withSplits:

  split long vectors into chunks of size splitSize

- splitSize:

  size of splits

## Value

list of vectors
